3. If log₂A + log₂(1/8) = 1, then the value of A is?
 Options:
 A. 8
B. 16
C. 32
D. 64
Answer: B. 16
Solution :
Step 1: Use logarithm property
log₂A + log₂(1/8) = log₂[A × (1/8)] = log₂(A/8)
Step 2: Substitute into equation
log₂(A/8) = 1
Step 3: Solve for A
A/8 = 2¹ = 2
A = 8 × 2 = 16
Step 4: Verification
log₂(16) + log₂(1/8) = 4 + (-3) = 1 
Therefore, A = 16.
4. If 4log P = 7log Q + 5, then the relation between P and Q is?
Options:
A. P⁴ = 100,000 × Q⁷
B. P² = 10,000 × Q⁷
C. P = 10Q
D. P⁴ = Q⁷ + 100,000
Answer: A. P⁴ = 100,000 × Q⁷
Solution :
Step 1: Apply logarithm power rule
4log P = log P⁴
7log Q = log Q⁷
Equation becomes: log P⁴ = log Q⁷ + 5
Step 2: Express 5 as a logarithm (base 10)
5 = 5 × log₁₀(10) = log₁₀(10⁵) = log(100,000)
log P⁴ = log Q⁷ + log 100,000
Step 3: Use logarithm addition property
log P⁴ = log(Q⁷ × 100,000)
(Since log a + log b = log(ab))
Step 4: Remove logarithms (antilog both sides)
P⁴ = 100,000 × Q⁷
Step 5: Verification
Taking log of both sides: log P⁴ = log(100,000 × Q⁷)
log P⁴ = log 100,000 + log Q⁷
log P⁴ = 5 + log Q⁷
4log P = 7log Q + 5 
Therefore, the relation between P and Q is: P⁴ = 100,000 × Q⁷

6. If log₆ P = Q, then log₂₁₆ P⁴ is equal to?
 Options:
 A. 4Q
B. Q/3
C. 8Q/3
D. (4/3)Q
Answer: D. (4/3)Q
Solution:
Step 1: Use the change of base formula
log₂₁₆ P⁴ = log₆ P⁴ / log₆ 216
Step 2: Simplify the numerator using power rule
log₆ P⁴ = 4 log₆ P
Since log₆ P = Q (given)
log₆ P⁴ = 4Q
Step 3: Simplify the denominator
We need to find log₆ 216
Note that 216 = 6³ (verify: 6 × 6 × 6 = 216)
Therefore: log₆ 216 = log₆ (6³) = 3
Step 4: Calculate the final result
log₂₁₆ P⁴ = 4Q / 3
log₂₁₆ P⁴ = (4/3)Q
Step 5: Verification
Given: P = 6^Q (from log₆ P = Q)
P⁴ = (6^Q)⁴ = 6^(4Q)
log₂₁₆(6^(4Q)) = 4Q × log₂₁₆ 6
= 4Q × (1/log₆ 216) = 4Q × (1/3) = (4/3)Q ✓
Therefore, log₂₁₆ P⁴ = (4/3)Q
7. 6^(log₂5) - 5^(log₂6) = ?
 Options:
A. log 125
B. 0
C. 3
D. 4
Answer: B. 0
Solution:
Step 1: Use the logarithm property: a^(log_b c) = c^(log_b a)
Step 2: Apply to the first term
6^(log₂5) = 5^(log₂6)
(This follows from the property with a=6, b=2, c=5)
Step 3: Substitute into the expression
6^(log₂5) - 5^(log₂6)
= 5^(log₂6) - 5^(log₂6)
= 0
Step 4: Verification using the property The property a^(log_b c) = c^(log_b a) can be proven:
Let x = a^(log_b c)
Taking log_b: log_b(x) = log_b(a^(log_b c)) = (log_b c)(log_b a)
Similarly, if y = c^(log_b a): log_b(y) = (log_b a)(log_b c)
Since multiplication is commutative: log_b(x) = log_b(y)
Therefore: x = y, so a^(log_b c) = c^(log_b a) 
Therefore, 6^(log₂5) - 5^(log₂6) = 0

By reciprocal property: log_D 2 = 1/log₂ D = 1/e
Step 4: Substitute and calculate
-3log_D 2 = -3(1/e) = -3/e

If log₂ D = e, then D = 2^e
log_D(1/8) = log_(2^e)(1/8) = log_(2^e)(2⁻³)
Using change of base: = log₂(2⁻³)/log₂(2^e) = -3/e 
Therefore, log_D(1/8) = -3/e

= log₈₀ 3 + log₈₀ 4 + log₈₀ 5
